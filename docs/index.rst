Bagpuss
===================================

.. image:: http://farm1.staticflickr.com/10/12460983_ff9f14e312.jpg
        :align: right
        :target: http://www.flickr.com/photos/weinhold71/12460983/

This is a reference implementation of a payment provider. It shows the API and
endpoints a payment provider should implement in order to integrate with the
Firefox Marketplace.

Bagpuss was also a children's TV show in the UK that a few of the developers
used to watch.
